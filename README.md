##  What Makes This Project Different?

This isn't another boring to-do app. It's a **personality quiz** that matches you to your coding language soulmate – built with personality, pixel art, and real-time data.

###  Design: Y2K + Frutiger Aero + Pixel Art

- **Y2K** – Pink-blue gradients, bold colors, Windows XP nostalgia
- **Frutiger Aero** – Glass-morphism, glossy buttons, depth and translucency
- **Retro Pixel Art** – Custom pixel icons I designed myself in Piskel

This is my **signature style** – playful, polished, and unforgettable.

###  Features

- **16 questions** across 4 categories: Python, TypeScript, JavaScript, Rust
- **Personalized results** based on your answers
- **Real-time GitHub API** – recommends the #1 trending repo for your result
- **Keyboard support** – press `Enter` to go to the next question
- **Responsive** – works perfectly on desktop AND mobile (custom backgrounds for both)
- **Confetti celebration** on results

###  Tech Stack

HTML · CSS · Vanilla JavaScript · GitHub REST API · Piskel · Canva
